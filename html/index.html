HTML

Lenguaje de marcado con el que declaramos la estructura de una pagina web, Consiste de elementos anidados y cada elemento tiene unos atributos que permiten modificar su comportamiento

DNS

Nos permite hablas con los servidores en internet sin necesidad de aprendernos sus direcciones IP

CSS

Es un lenguaje de estilos que nos permite definir como queremos que se vea nuestra pagina. CSS se encarga de definir como debe ser renderizado el elemento estructurado en la pantalla

TCP/IP

TCP/IP (Transmision Control Protocol/Internet Protocol) Es un conjunto de protocolos de red que permiten la comunicacion entre dispositivos informaticos, incluyendo ordenadores y otros dispositivos , en una red o entre redes, incluyendo la internet.
